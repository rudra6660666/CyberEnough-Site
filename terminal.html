Kali Linux Terminal

Practice Linux commands in a simulated Kali Linux environment. Try commands like ls, pwd, whoami, or cybersecurity tools like nmap.

Terminal - kali@kali:~
┌──(kali㉿kali)-[~]
└─$ Welcome to Kali Linux Test Terminal!
Type 'help' for available commands or try basic Linux commands.
┌──(kali㉿kali)-[~]
└─$ _

Available Commands

Basic Linux

  • ls - List directory contents
  • cd - Change current directory
  • pwd - Print working directory
  • whoami - Current user
  • uname -a - System information
  • cat /etc/passwd - View users

Network Tools

  • nmap -sn 192.168.1.0/24 - Network scan
  • netstat -tuln - Network connections
  • ifconfig - Network interfaces
  • ping google.com - Network connectivity

Security Tools

  • nikto -h target.com - Web vulnerability scanner
  • dirb http://target.com - Directory brute force
  • sqlmap --help - SQL injection tool
  • hydra -l admin -P passwords.txt ssh://target - Password cracking
